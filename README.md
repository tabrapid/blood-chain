# Blood Chain

Blood Chain — bu donorlar, qon markazlari va shifoxonalar o‘rtasida qon donatsiyasi jarayonini raqamlashtirish va boshqarish uchun yaratilgan zamonaviy platforma.

## Umumiy maqsad
- Donorlarni jalb qilish va rag‘batlantirish
- Qon markazlari va shifoxonalar o‘rtasida tezkor axborot almashinuvi
- Qon zaxiralarini boshqarish va monitoring qilish
- Jamiyatda donorlik madaniyatini rivojlantirish

## Loyihaning asosiy modullari
- Web ilova (Next.js, TypeScript, Tailwind CSS)
- Supabase va Prisma asosidagi backend
- Ma’lumotlar bazasi sxemasi va migratsiyalar (prisma, supabase)

## Qisqacha tuzilma
- `web/` — asosiy frontend va backend kodlari
- `supabase/` — ma’lumotlar bazasi sxemasi va SQL migratsiyalar

## Boshlash
1. `web/` papkasiga o‘ting va README.md bo‘yicha ishga tushiring
2. Supabase yoki SQLite konfiguratsiyasini sozlang
